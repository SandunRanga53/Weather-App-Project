# 🌦️ Weather App Project

![GitHub repo size](https://img.shields.io/github/repo-size/SandunRanga53/Weather-App-Project)
![GitHub stars](https://img.shields.io/github/stars/SandunRanga53/Weather-App-Project?style=social)
![GitHub forks](https://img.shields.io/github/forks/SandunRanga53/Weather-App-Project?style=social)

A simple and responsive **Weather Application** that allows users to search for the current weather conditions of any city.
The application fetches real-time weather data from a weather API and displays important information such as temperature, weather conditions, and location.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Display current temperature
* ☁️ Show weather conditions (Cloudy, Rainy, Sunny, etc.)
* 📍 Show city/location information
* 📱 Responsive design

## ⚙️ Installation

Clone the repository:

git clone https://github.com/SandunRanga53/Weather-App-Project.git

Go to the project folder:

cd Weather-App-Project

Open the project:

Open `index.html` in your browser.

---

## 🌐 API Used

This project uses the **OpenWeather API** to fetch real-time weather data.

Example request:

https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}

Get your free API key here:
https://openweathermap.org/api
